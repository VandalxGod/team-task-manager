# 🚀 Team Task Manager (MERN Stack)

## 📌 Features
- Authentication (JWT)
- Role-Based Access (Admin / Member)
- Project Management
- Multi-user Task Assignment
- Task Status Tracking
- Dashboard Analytics

## 🛠 Tech Stack
- Frontend: React + Tailwind
- Backend: Node.js + Express
- Database: MongoDB

## 🔐 Roles
- Admin: Full access
- Member: Limited access

## 🌐 API Routes
- /api/auth
- /api/projects
- /api/tasks
- /api/dashboard

## 🚀 How to Run

### Backend
```bash
cd backend
npm install
npm run dev
